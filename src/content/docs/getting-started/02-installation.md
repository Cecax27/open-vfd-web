## Requirements

- A modern desktop operating system
- Access to official OpenVFD release files
- Network access if you plan to use OPC UA connectivity

## Install steps

1. Open the [Downloads](/downloads) page.
2. Pick the latest stable release.
3. Download the artifact for your platform.
4. Unpack or install according to your operating system conventions.

## Verify the installation

- Launch OpenVFD
- Confirm the simulation panel renders
- Check that parameter fields are editable
