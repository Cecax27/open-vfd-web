## First simulation workflow

1. Open OpenVFD and load default parameters.
2. Set a target frequency and ramp time.
3. Add an initial motor load value.
4. Start the simulation and observe output variables.

## What to watch

- Speed response during acceleration
- Current or torque trend under load
- Stability when changing frequency commands
