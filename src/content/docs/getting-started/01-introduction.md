OpenVFD is an open source simulator focused on variable frequency drive behavior and motor response modeling.

## What you can simulate

- Motor load behavior under changing operating conditions
- Inertia effects that influence ramp-up and deceleration
- Basic VFD parameters to observe control response
- OPC UA variable exchange for controls and telemetry

## Typical use cases

- Validate tuning assumptions before field deployment
- Build training examples for VFD control concepts
- Prototype SCADA integration flows with OPC UA endpoints
