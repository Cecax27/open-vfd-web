## Execution cycle

1. Confirm initial parameters.
2. Start simulation.
3. Observe trend behavior and telemetry values.
4. Stop simulation after reaching a stable region.

## Repeatability tips

- Keep a baseline parameter set
- Record which variable changed per run
- Compare stable-state results before and after changes
