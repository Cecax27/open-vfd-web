## Core parameter groups

- Output frequency target
- Acceleration and deceleration ramps
- Voltage reference behavior

## Tuning approach

1. Apply a baseline profile.
2. Change one parameter at a time.
3. Compare trend changes in speed and load response.

This one-change-at-a-time strategy avoids false assumptions and keeps tuning reproducible.
